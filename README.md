# Movimiento Browniano en un Campo Magnético

Este repositorio contiene la implementación en Python de la simulación de la ecuación de Langevin para una partícula browniana cargada en un campo magnético uniforme.

Ejecutar en la terminal:

DinamicadelaVelocidadBrowniano.py

## Parámetros utilizados

m = 1  
beta = 1  
q = 1  
B = 2  
T = 1  
dt = 0.001  

donde:

- m es la masa de la partícula  
- beta es el coeficiente de fricción  
- q es la carga  
- B es el campo magnético aplicado en la dirección z  
- T es la temperatura  
- dt es el paso temporal de integración numérica
